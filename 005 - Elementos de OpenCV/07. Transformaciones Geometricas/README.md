# 📐 Transformaciones Geométricas con OpenCV 🖼️

¡Hola! Bienvenido a este laboratorio interactivo sobre **transformaciones geométricas de imágenes** usando Python y OpenCV. 🚀

Este repositorio contiene un notebook de Jupyter (`.ipynb`) donde exploramos cómo manipular la geometría de las imágenes. Aprenderás a escalar, mover, rotar y deformar imágenes, habilidades esenciales para cualquier proyecto de Visión por Computadora, desde la corrección de imágenes hasta la realidad aumentada.

## 🌟 ¿Qué aprenderás aquí?

En este notebook, cubrimos las transformaciones geométricas más importantes, paso a paso:

* **✂️ Recorte y Cambio de Perspectiva:** Aprende a seleccionar una región de una imagen (como una carta de una baraja) y a transformarla para que parezca que la estás viendo de frente.
* **🔍 Escalar (Scaling):** Cambia el tamaño de una imagen, haciéndola más grande o más pequeña.
* **↔️ Traslación (Translation):** Mueve una imagen a una nueva posición dentro del lienzo.
* **🔄 Rotación (Rotation):** Gira una imagen alrededor de un punto central con cualquier ángulo que desees.
* **✨ Transformación Afín (Affine Transform):** Realiza transformaciones más complejas como cizallamiento (shear), combinando rotación, escalado y traslación.
* ** warping perspectiva (Perspective Warp):** Corrige la distorsión de la perspectiva, ideal para escanear documentos o reconocer objetos en ángulo.
